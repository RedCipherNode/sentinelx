# Scope

## In Scope

* Static analysis
* Files
* Archives
* Projects
* URLs
* Malware indicators
* Scam indicators
* Phishing indicators
* Risk assessment
* Explainable findings

---

## Out of Scope

* Dynamic analysis
* Sandbox execution
* Antivirus
* Malware removal
* Process monitoring
* Memory analysis
* Network monitoring
* Endpoint protection

---

## Principles

* Never execute untrusted content.
* Evidence before conclusions.
* Explain every finding.
* Prefer deterministic analysis over guesswork.
